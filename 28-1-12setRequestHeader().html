JavaScript 中，Content-Type 是一个 HTTP 头，它用于指示资源的媒体类型。媒体类型是一个字符串，它表示了文件的格式。例如，对于图像文件，它的媒体类型可能是 image/png 或 image/jpg 等

在响应中，Content-Type 头告诉客户端返回的内容的实际类型。这个头的值可能会被忽略，例如当浏览器进行 MIME 探测时；可以设置 X-Content-Type-Options 头的值为 nosniff 来防止这种行为

在请求中（例如 POST 或 PUT ），客户端告诉服务器实际发送的数据的类型
你可以使用 Content-Type 头来指定你发送或接收的 JSON 数据的格式，例如：
Content-Type: application/json
这样，服务器就可以正确地解析和处理 JSON 数据了。
