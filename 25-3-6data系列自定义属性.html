自定义data属性是一种HTML5新增的特性，用于在标准的HTML元素上存储和获取一些额外的信息或数据，而不需要使用非标准的属性或DOM上的额外属性。

自定义data属性的用法是：
在HTML元素的开始标签中定义一个以data-开头的属性，并赋予一个值。例如：<div data-color="red">
在JavaScript中，通过dataset属性来设置或获取自定义data属性。例如：$0.dataset.color = "red"; 或 var color = $0.dataset.color;
在CSS中，通过attr()函数或属性选择器来访问或改变自定义data属性。例如：div::before { content: attr(data-color); } 或 div[data-color="red"] {}

自定义data属性有一些使用案例，例如：
存储一些常变化的数据，如游戏中的分数
存储一些与特定元素相关联但没有明确定义的数据
存储一些不需要显示或访问的数据
