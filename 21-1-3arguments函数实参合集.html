arguments对象是一个伪数组(类数组对象)，它可以在函数内部访问传递给函数的参数的值 它有一个length属性，表示参数的个数，以及一个索引属性，表示每个参数的值。例如，如果一个函数传递了三个参数，你可以这样访问它们：

  • arguments[0]; // 第一个参数
  • arguments[1]; // 第二个参数
  • arguments[2]; // 第三个参数

    • 你也可以给索引属性赋值，改变参数的值： arguments[1] = '新值';

    arguments对象对于处理变数量的参数的函数很有用，例如Math.min()。你可以使用arguments.length来确定传递给函数的参数的个数，然后使用arguments对象来处理每个参数。

    注意，在严格模式下，或者在使用剩余参数、默认参数或解构赋值参数的函数中，arguments对象不会与实际的参数同步更新。也就是说，改变arguments对象中的值不会影响实际的参数，反之亦然 如果你想把arguments对象转换为真正的数组，你可以使用Array.from()方法或扩展运算符：

  • let args = Array.from(arguments);
  • let args = [...arguments];
