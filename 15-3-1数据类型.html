JavaScript的数据类型是指变量可以存储的数据的种类。在编程中，数据类型是一个重要的概念，因为不同的数据类型有不同的特性和操作¹。

Number：表示数字，包括整数和浮点数。例如：`var x = 3.14;`

BigInt：表示大于253-1或小于-253的整数。例如：`var y = 9007199254740991n;`

String：表示文本，由一系列字符组成。例如：`var z = "Hello";`

Boolean：表示逻辑值，只有两个可能的值：true或false。例如：`var a = true;`

Null：表示一个空值，只有一个可能的值：null。例如：`var b = null;`

Undefined：表示一个未定义的值，只有一个可能的值：undefined。例如：`var c; // c is undefined`

Symbol: 表示一个唯一且不可变的标识符，用于创建对象属性。例如: `var d = Symbol("id");`

Object: 表示一个复杂的数据结构，由键值对或其他属性组成。例如: `var e = {name: "Alice", age: 25};`

除了这些基本数据类型外，JavaScript还支持一种轻量级的数据交换格式，叫做JSON（JavaScript Object Notation）²。
JSON可以用来构建通用的数据结构，并在不同环境和语言之间传输数据。
